# Sudoku solver

## Installation

To install sudoku solver type the following in the terminal:

```bash
pip install sudoku-solver
```

## Usage

To start sudoku-solver type the following in your terminal:

```bash
sudoku-solver
```

For help about arguments type:

```bash
sudoku-solver --help
```
